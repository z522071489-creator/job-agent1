    配置项值

    地域华北地区（北京）

    内存512 MB
    超时时间600 秒（必须设最大值）

2
配置 GitHub Secrets

进入 GitHub 仓库 → Settings → Secrets and variables → Actions → New repository secret

Secret 名称值说明
SCF_URL SCF API网关触发器URL 从腾讯云控制台复制
SERVERCHAN_KEY 你的 SendKey（SCTxxxxxxx...） Server酱推送微信
✅ GITHUB_TOKEN 不需要手动设置，Actions 会自动注入。

常见问题 & 调优建议
问题解决方案
SCF 函数超时 减少公众号数量，或把微信采集和网站采集分成两个 SCF 函数
搜狗返回验证码 增加 time.sleep 间隔（8-15秒），腾讯云北京IP一般不被封
Kimi 余额不足 改用 moonshot-v1-8k 更便宜，或把AI分析移到 GitHub Actions 端
网站改版后采集为空 更新 scf_function/index.py 中对应采集函数的 CSS 选择器
Excel 下载需要登录 将 GitHub 仓库设为 Public，或在 Actions Artifacts 下载
